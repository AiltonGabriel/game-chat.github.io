GAME - CHAT

O Game-Chat é uma ferramenta para troca de mensagens a respeito de um certo game.

Desenvolvido por: Ailton Gabriel de Oliveira

  • Esta ferramenta criada como Trabalho Prático para disciplina Desenvolvimento de Páginas Web, do Curso Superior de Tecnologia em Sistemas para Internet no If Sudeste MG - Campus Barbacena
  • Tecnologias utilizadas:
    • HTML (HyperText Markup Language), CSS (Cascading Style Sheets), e JavaScript.
    • Também foi utilizada a plataforma Firebase do Google.
    • Todas as informações a respeito dos games foram fornecidas pela Web API do Steam.
